Extreme Value Theorem

If f(x)f(x)f(x) is continuous on a closed interval [a,b][a,b][a,b], then fff must attain a maximum and a minimum, each at least once.

f′(c)f'(c)f′(c) is a critical point of fff if f′(c)=0f'(c) = 0f′(c)=0 or f′(c)=undefinedf'(c) = undefinedf′(c)=undefined

First Derivative Test

f(x)f(x)f(x) will have a relative max when f′(x)f'(x)f′(x) switches from positive to negative around a critical point.

f(x)f(x)f(x) will have a relative min when f′(x)f'(x)f′(x) switches from negative to positive around a critical point.

  • Relative extremas are greater/less than their surround points.
  • Endpoints are not relative extremas. There are no points on one side of the endpoint.
  • To find relative extremas, find all critical points ccc and check if f′f'f′ changes signs around point ccc.

Concavity

A function f is concave up when the slope increases. The rate of change of the function is increasing. The graph opens up.

  • f′′(x)>0f''(x) > 0f′′(x)>0

A function f is concave down when the slope decreases. The rate of change of the function is decreasing. The graph opens up.

  • f′′(x)<0f''(x) < 0f′′(x)<0

Inflection Point

A point c is an inflection point if the graph of fff changes concavity around x=cx=cx=c.

f′′(c)=0f''(c) = 0f′′(c)=0
  • f′′(c)f''(c)f′′(c) has to change signs around ccc.

Second Derivative Test

A critical point ccc is a relative max if f′′(c)<0f''(c) < 0f′′(c)<0.

A critical point ccc is a relative min if f′′(c)>0f''(c) > 0f′′(c)>0.

  • If the graph was concave up around a critical point, then the slope is increasing, therefore f′(x)f'(x)f′(x) is going from negative to positive.
  • If the graph was concave down around a critical point, then the slope is decreasing, therefore f′(x)f'(x)f′(x) is going from positive to negative.

Mean Value Theorem

If fff is a continuous function on the closed interval [a,b][a,b][a,b] and differentiable on the open interval (a,b)(a,b)(a,b), then there exists a point ccc in (a,b)(a,b)(a,b) such that

  • The MVT requires the function be continuous and differentiable.
    • fff is only differentiable at an open interval because it is possible the endpoints to not be differentiable.
  • The theorem can be restated as the secant between two points is parallel to a tangent line within the interval.
  • For questions asking for a justification of the MVT, it must be stated that the function is differentiable.
    • Ex: Since fff is differentiable over (a,b)(a,b)(a,b), by the MVT, there is some ccc in (a,b)(a,b)(a,b) where f′(c)=f(b)−f(a)b−af'(c) = \frac{f(b)-f(a)}{b-a}f′(c)=b−af(b)−f(a)​.

Unit 6: Differential Equations

A differential equation is an equation that involves at least 1 derivative

A solution to a differential equation is any function that satisfies the equation.

The paricular solution of a differential equation is the original function in the form of y=f(x)y=f(x)y=f(x) that contains a specific point, referred to as the initial condition.

The domain of the particular solution is the interval over which f(x)f(x)f(x) is defined, differentiable, and contains the initial condition.

Separable Differential Equation

A differential equation that can be written with each of the 2 variables on a different side of the equation.

  1. Separation of variables is rewriting the equation such that the y terms are on one side and the x terms on the other (including dydydy and dxdxdx).
  2. Integrate both sides with the constant of integration, ccc, on the side with the x terms.
  3. For particular solutions, ccc needs to found.
  4. Solve for yyy to find the solution to the differential equation.

Slope Fields

Short line segments with slope dydx\frac{dy}{dx}dxdy​ at (x,y)(x,y)(x,y), form a slope field and shows the general shape of all solutions of dydx=f(x)\frac{dy}{dx} = f(x)dxdy​=f(x).

To match slope fields with the correct dydx\frac{dy}{dx}dxdy​ equation, look at the equation.

  1. If dydx\frac{dy}{dx}dxdy​ only contains either xxx or yyy, then it will only change as that variable is changed.
    • Ex: dydx=x2\frac{dy}{dx} = x^2dxdy​=x2 will only change with x so all line segments on x=2x=2x=2 will be the same.
  2. dydx\frac{dy}{dx}dxdy​ may contain a variable raised to an even power. This variable will not change the sign of the slope.
    • Ex: dydx=xy2\frac{dy}{dx} = \frac{x}{y^2}dxdy​=y2x​ Since yyy is squared, then the slopes should be negative when xxx is negative and vice versa.
  3. Notice places where dydx\frac{dy}{dx}dxdy​ is either 0 or undefined. That is the slope is either horizontal or vertical. If the slopes along a given xxx or yyy is horizontal/vertical, that should be a factor in the differential equation.
    • Ex: All the slopes along y=2y=2y=2 is horizontal and vertical along x=1x=1x=1. This means the differential equation should have (y−2)(y-2)(y−2) in the numerator and (x−1)(x-1)(x−1) in the denominator.
  4. Pick points that are unique to each graph/differential equation. Find dydx\frac{dy}{dx}dxdy​ for that point and eliminate some graphs/equations. Repeat as necessary.

Notes

  • When performing separation of variables, terms should not be added/subtracted. Only move the terms using multiplication/division.

For areas bounded by 2 functions, there will be two intersection points. The two points where the two functions intersect are the limits of integration.
